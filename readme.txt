Prevent visitors from accessing WooCommerce-pages and contents.

== Description ==
This plugin will make all WooCommerce content inaccessible to visitors. All requests to WooCommerce-pages e.g. to certain category, will be redirected to my-account page where you can implement your preferred registration. Also plugin hides prices from products and replaces them with note about registration required. 

== Installation ==
* Install WooCommerce 
* Install WooCommerce Require Registered User
* Go to WooCommerce -> Settings -> Products -> Require Registered User
* Click \"Enable Require Registered User\"
* Save settings
